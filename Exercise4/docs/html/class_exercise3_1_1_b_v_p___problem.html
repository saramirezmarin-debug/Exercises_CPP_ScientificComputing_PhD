Detailed Description

Test problem used in the assignment text.

Sistema continuo:

  • \(-u'' + 2u - v = f_1\)
  • \(-v'' - u + 2v = f_2\)

with exact solution:

  • \(u(x) = \sin(\pi x)\)
  • \(v(x) = \sin(2\pi x)\)

The boundary conditions are homogeneous, but they are still expressed through exact(a,·) and exact(b,·) so that the code stays consistent with more general cases.
