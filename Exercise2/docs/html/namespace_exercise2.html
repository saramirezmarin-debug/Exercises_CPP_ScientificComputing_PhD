◆ run_experiment()

Experiment_Result Exercise2::run_experiment ( BVP_Problem_base const & problem,
integer N )
static

Run a complete experiment for a given grid size N.

Steps:

  • build the uniform grid;
  • assemble the block tridiagonal matrix;
  • inject boundary contributions into the right-hand side;
  • solve the linear system;
  • compute the error, when the exact solution is available.

Nnumber of interior nodes.

void Exercise2::tic ( )

Start a lightweight global timer.

Save the current time as the initial reference.

It is meant for simple local timing measurements: tic(); ...; toc();

The implementation is intentionally minimal: it is only used to measure the cost of the linear solve in the test program.
