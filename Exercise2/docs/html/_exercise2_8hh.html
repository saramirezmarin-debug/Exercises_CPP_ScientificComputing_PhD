Detailed Description

Data structures and interfaces for the Exercise2 problem.

This file contains:

  • Eigen-based numeric aliases;
  • a block tridiagonal linear solver;
  • an abstract base class for problems of the form \(-Y''(x) + A(x)Y(x) = F(x)\).

Unicode symbols are intentionally used in the comments to make the relation between mathematics and implementation clearer.
