real_type h )

One Runge-Kutta 4 integration step.

Computes x_{k+1} from x_k using:

k1 = f(t, x) k2 = f(t+h/2, x+h*k1/2) k3 = f(t+h/2, x+h*k2/2) k4 = f(t+h, x+h*k3)

x_{k+1} = x_k + h/6 * (k1 + 2k2 + 2k3 + k4)
