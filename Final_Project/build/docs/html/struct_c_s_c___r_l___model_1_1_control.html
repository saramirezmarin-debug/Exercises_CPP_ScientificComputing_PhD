Detailed Description

template<typename T>
struct CSC_RL_Model::Control< T >

Algebraic variables produced by the PLL and control system.

This structure stores intermediate control variables. These variables are computed from the current state and references, but they are not dynamic states themselves.

The PLL frequency estimate is

\[\hat{\omega} = \omega_0 + k_{p,\mathrm{pll}} e_{\mathrm{pll}} + k_{i,\mathrm{pll}} \xi_{\mathrm{pll}}. \]

The outer loop generates the active-power reference:

\[P_{\mathrm{ref}} = k_{pO} \left(i_{dc,\mathrm{ref}}^2-i_{stk}^2\right) + k_{iO}\xi_{\mathrm{idc2}}. \]

Template Parameters
TScalar type used for the algebraic variables.
