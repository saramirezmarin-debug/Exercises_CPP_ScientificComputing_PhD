Detailed Description

Evaluator of the CSC-RL equations and control laws.

All physical parameters and controller gains are supplied through CSC_RL_Parameters. The model itself stores no internal state.

Computes PLL, outer-loop, current-loop, and voltage-loop variables.

The PLL error is

\[e_{\mathrm{pll}}=\frac{e_q}{V_{\mathrm{dq,nom}}}. \]

The estimated angular frequency is

\[\hat{\omega} = \omega_0 + k_{p,\mathrm{pll}} e_{\mathrm{pll}} + k_{i,\mathrm{pll}} \xi_{\mathrm{pll}}. \]

The outer DC-current loop uses

\[E_{\mathrm{idc2}} = i_{dc,\mathrm{ref}}^2 - i_{stk}^2, \]

\[P_{\mathrm{ref}} = k_{pO}E_{\mathrm{idc2}} + k_{iO}\xi_{\mathrm{idc2}}. \]

The AC current references are obtained from the synchronous-frame power equations:

\[i_d^* = \frac{ e_dP_{\mathrm{ref}}+e_qQ_{\mathrm{ref}} }{ e_d^2+e_q^2 }, \]

\[i_q^* = \frac{ e_qP_{\mathrm{ref}}-e_dQ_{\mathrm{ref}} }{ e_d^2+e_q^2 }. \]

Evaluates the nonlinear right-hand side of the CSC-RL model.

This function evaluates

\[\frac{dx}{dt}=f(t,x). \]

The grid-side current equations are

\[\frac{d i_{gd}}{dt} = \frac{ e_{gd} - R_g i_{gd} - e_d + \hat{\omega}L_g i_{gq} }{L_g}, \]

\[\frac{d i_{gq}}{dt} = \frac{ e_{gq} - R_g i_{gq} - e_q - \hat{\omega}L_g i_{gd} }{L_g}. \]

The PCC voltage equations are

\[\frac{d e_d}{dt} = \frac{ i_{gd} - i_d + \hat{\omega}C_g e_q }{C_g}, \]

\[\frac{d e_q}{dt} = \frac{ i_{gq} - i_q - \hat{\omega}C_g e_d }{C_g}. \]

The PWM-CSC equations are

\[\frac{d i_d}{dt} = \frac{ e_d - v_d - R_f i_d + \hat{\omega}L_f i_q }{L_f}, \]

\[\frac{d i_q}{dt} = \frac{ e_q - v_q - R_f i_q - \hat{\omega}L_f i_d }{L_f}, \]

\[\frac{d v_d}{dt} = \frac{ i_d - m_d i_{stk} + \hat{\omega}C_f v_q }{C_f}, \]

\[\frac{d v_q}{dt} = \frac{ i_q - m_q i_{stk} - \hat{\omega}C_f v_d }{C_f}, \]

\[\frac{d i_{stk}}{dt} = \frac{ m_dv_d + m_qv_q - R_L i_{stk} }{L_{dc}}. \]
