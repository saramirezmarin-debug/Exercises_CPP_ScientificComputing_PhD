Detailed Description

Generic interface for Newton-based equilibrium calculation.

This file defines the reference and option structures used to compute a steady-state operating point before running the time-domain simulation.

A model-specific equilibrium problem must derive from AD::NewtonProblemAD<ODE::real_type> and define the nonlinear residual:

\[F(z)=0. \]

Solves a generic automatic-differentiation Newton equilibrium problem.

The function returns false if the nonlinear residual does not reach the requested tolerance. In that case, the simulation can continue using manual initial conditions.
